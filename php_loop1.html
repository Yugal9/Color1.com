Using for and foreach Loops with Numeric Array

In this example, we have a numeric array called $fruits. We use a for loop to iterate through the array by index and a foreach loop to iterate through the array directly.

Program


// Numeric Array
$fruits = array("Apple", "Banana", "Cherry", "Date");
// Using a for loop to iterate through the array
echo "Using for loop:< br>";
for ($i = 0; $i < count($fruits); $i++) {
echo $fruits[$i] . " ";
}
echo "< br>";
// Using a foreach loop to iterate through the array
echo "Using foreach loop:< br>";
foreach ($fruits as $fruit) {
echo $fruit . " ";
}
?>

OUTPUT


Using for loop:
Apple Banana Cherry Date
Using foreach loop:
Apple Banana Cherry Date
