Retrieving and Using a Cookie

In this example, we check if the "user" cookie exists using isset($_COOKIE["user"]). If it exists, we retrieve its value and display a personalized message. If it doesn't exist, we display a generic "Welcome, guest!" message.

Program


// Check if the cookie exists
if (isset($_COOKIE["user"])) {
$user = $_COOKIE["user"];
echo "Welcome back, " . $user . "!";
} else {
echo "Welcome, guest!";
}
?>

OUTPUT


If the "user" cookie exists in the user's browser and has not expired, the output will be:
Welcome back, JohnDoe!
If the "user" cookie does not exist, the output will be:
Welcome, guest!
