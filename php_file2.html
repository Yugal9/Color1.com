Reading a file

In this example, we check if the file "example.txt" exists, and if it does, we open it in read mode and display its content using fgets.

Program


// File path
$filename = "example.txt";
// Check if the file exists
if (file_exists($filename)) {
// Open the file in read mode
$file = fopen($filename, "r");
// Read and display the file content
if ($file) {
while (!feof($file)) {
echo fgets($file);
}
fclose($file);
} else {
echo "Unable to open the file for reading.";
}
} else {
echo "The file does not exist.";
}
?>

OUTPUT


This is some text that we are writing to the file.
