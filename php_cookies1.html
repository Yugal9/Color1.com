Setting a Cookie

In this example, we'll set a cookie with a name, value, and expiration time. The code below sets a cookie named "user" with the value "JohnDoe" that expires in one hour. In this example, the setcookie function is used to set a cookie.

Program


// Set a cookie with a name, value, and expiration time
$cookie_name = "user";
$cookie_value = "JohnDoe";
$expiration_time = time() + 3600; // 1 hour from the current time
setcookie($cookie_name, $cookie_value, $expiration_time, "/");
?>
