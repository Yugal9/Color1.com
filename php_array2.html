Associative Array

In this example, we create an associative array called $person with key-value pairs representing information about a person. We use the print_r function to display the contents of the associative array.

Program


// Define an associative array with key-value pairs
$person = array(
"name" => "John",
"age" => 30,
"city" => "New York"
);
// Output the array using print_r
echo "Associative Array:< br>";
print_r($person);
?>

OUTPUT


Associative Array:
Array
(
[name] => John
[age] => 30
[city] => New York
)
