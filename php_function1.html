A Function to Calculate the Square of a Number

In this example, we define a function called calculateSquare that takes a number as an argument, calculates its square, and returns the result. We then call this function with the argument 5 and display the result.

Program


// Define a function to calculate the square of a number
function calculateSquare($number) {
$square = $number * $number;
return $square;
}
// Call the function
$result = calculateSquare(5);
// Output the result
echo "The square of 5 is: " . $result;
?>

OUTPUT


The square of 5 is: 25
