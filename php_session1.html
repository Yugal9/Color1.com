Creating and Using Sessions

In this example, we start a session, set session variables to store user information, retrieve the session data, and then end the session.

Program


// Start a session
session_start();
// Set session variables
$_SESSION["username"] = "JohnDoe";
$_SESSION["email"] = "john@example.com";
// Retrieve and display session data
echo "Welcome, " . $_SESSION["username"] . "! Your email is " . $_SESSION["email"];
// End the session
session_destroy();
?>

OUTPUT


Welcome, JohnDoe! Your email is john@example.com
