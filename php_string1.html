String Concatenation

In this PHP program, we create two string variables, $string1 and $string2. We then concatenate them using the . operator to create a new string, $combinedString. Finally, we use the strlen function to find the length of the combined string and display it.

Program


// Create a string variable
$string1 = "Hello, ";
// Concatenate two strings
$string2 = "World!";
$combinedString = $string1 . $string2;
// Find the length of the string
$length = strlen($combinedString);
// Output the result
echo "Combined String: " . $combinedString . "
"; echo "Length of the Combined String: " . $length;
?>

OUTPUT


Combined String: Hello, World!
Length of the Combined String: 13
