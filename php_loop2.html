Using while and do...while Loops to Count Down

In this example, we use a while loop to count down from 5 to 1, and then a do...while loop to do the same. Both loops print the numbers from 5 to 1..

Program


// Using a while loop to count down from 5 to 1
echo "Using while loop:< br>";
$counter = 5;
while ($counter >= 1) {
echo $counter . " ";
$counter--;
}
echo "< br>";
// Using a do...while loop to count down from 5 to 1
echo "Using do...while loop:< br>";
$counter = 5;
do {
echo $counter . " ";
$counter--;
} while ($counter >= 1);
?>

OUTPUT


Using while loop:
5 4 3 2 1
Using do...while loop:
5 4 3 2 1
