Checking for an Active Session

In this example, we start or resume a session, check if the user is logged in based on the presence of session data, and display different messages accordingly.

Program


// Start or resume a session
session_start();
// Check if the user is logged in
if (isset($_SESSION["username"])) {
echo "Welcome back, " . $_SESSION["username"] . "!";
} else {
echo "Please log in.";
}
// End the session
session_destroy();
?>

OUTPUT


When a session is active:
Welcome back, JohnDoe!
When no session is active:
Please log in.
