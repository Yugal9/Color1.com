Numeric Array

In this example, we create a numeric array called $fruits with four elements. We then use the print_r function to display the contents of the array.

Program


// Define a numeric array with some elements
$fruits = array("Apple", "Banana", "Cherry", "Date");
// Output the array using print_r
echo "Numeric Array:< br>";
print_r($fruits);
?>

OUTPUT


Numeric Array:
Array
(
[0] => Apple
[2] => Cherry
[1] => Banana
[3] => Date
)
