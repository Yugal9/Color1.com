Writign to a file.

In this example, we open a file called "example.txt" in write mode, write some text to it using fwrite, and then close the file.

Program


// File path
$filename = "example.txt";
// Open the file in write mode
$file = fopen($filename, "w");
// Check if the file was opened successfully
if ($file) {
// Content to write
$content = "This is some text that we are writing to the file.\n";
// Write the content to the file
fwrite($file, $content);
// Close the file
fclose($file);
echo "File written successfully!";
} else {
echo "Unable to open the file.";
}
?>

OUTPUT


File written successfully!
This is some text that we are writing to the file.
