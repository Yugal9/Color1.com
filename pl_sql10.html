Iterative Control Statements

OPL/SQL Iterative control statements allow a programmer to execute a set of statements multiple times. All iterative statements must have a conditional statement to check whether to continue to execute the block of statement another time.
Simple Loop- LOOP- END LOOP;
WHILE loop- WHILE-LOOP-END LOOP;
FOR Loop- FOR-IN-LOOP-END LOOP;

Simple Loop

The simple loop is infinite. The loop keeps on executing the enclosed statements infinitely. Simple loop needs an exit condition and exit statement to terminate.

Syntax

LOOP
Loop statements
Conditional exit using EXIT|EXIT WHEN
END LOOP;
