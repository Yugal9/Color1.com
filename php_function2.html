A Function to Generate a Random Number

In this example, we define a function called generateRandomNumber that generates a random number between 1 and 10 using the rand function. We then call this function and display the generated random number. The specific random number you see in the output will vary each time you run the script.

Program


// Define a function to generate a random number between 1 and 10
function generateRandomNumber() {
$randomNumber = rand(1, 10);
return $randomNumber;
}
// Call the function
$random = generateRandomNumber();
// Output the result
echo "Random number between 1 and 10: " . $random;
?>

OUTPUT


Random number between 1 and 10: [A random number between 1 and 10]
